# Formulario de Registro de Eventos con POST

## Descripción
Este proyecto es un formulario completo de registro para eventos que utiliza el método tradicional de envío POST. Cuando el usuario envía el formulario, la página se recarga y muestra los datos procesados.

## Cómo usar

### Requisitos previos:
- XAMPP (o cualquier servidor con Apache y PHP)
- Navegador web moderno

### Instalación:
1. Copia la carpeta `creacion-formularios` en `C:\xampp\htdocs\`
2. Inicia Apache desde el panel de control de XAMPP
3. Abre tu navegador y ve a: `http://localhost/creacion-formularios/front/formulario.html`

### Flujo de datos:
```
Usuario rellena formulario → Click "Enviar" → 
Validación JavaScript → Envío POST → 
PHP procesa datos → Devuelve página HTML completa → 
Navegador muestra nueva página
```
